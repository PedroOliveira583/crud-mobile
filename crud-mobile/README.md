# CRUD Mobile

Aplicativo React Native com Expo para gerenciamento de produtos.

## Instalação

```bash
npm install
```

## Configuração

Certifique-se de que o backend está rodando em `http://localhost:3000`.

Para testar em dispositivo físico, altere o `API_URL` em `api.js` para o IP da sua máquina.

## Execução

```bash
npx expo start
```

Escaneie o QR Code com o aplicativo Expo Go (Android) ou Camera (iOS).

## Funcionalidades

- Listar produtos
- Criar novo produto
- Editar produto existente
- Excluir produto
- Validação de formulários
